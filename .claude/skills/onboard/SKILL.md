# Onboard Skill
Orientation at the start of a new session. Establishes current project state.

1. Run `git log --oneline -10` to see recent commits
2. Run `git status` to see any in-progress or uncommitted work
3. Read `README.md` for a project overview
4. Read `~/.claude/projects/-Users-markrubin-Development-bayclubconnect-helper/memory/MEMORY.md` to recall notes from prior sessions
5. Briefly summarize:
   - What the project is (from README)
   - What was last worked on (from git log + memory)
   - Any uncommitted changes or in-progress work
   - Any open questions or planned next steps noted in memory
6. Ask: "What would you like to work on today?"
